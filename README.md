# Instruções

Código desenvolvido visando ser utilizado em uma placa de desenvolvimento BitDogLab.<br>
Coloque o RP2040 em modo bootsel manualmente e carregue o programa. Com ele carregado, o botão B também faz essa função.<br>
Girar o joystick altera a intensidade dos LEDs vermelho (Eixo X) e azul (Eixo Y). Acionar o botão do Joystick liga o LED verde e cria uma borda no display LCD, indicando seu acionamento.<br>
Acionar o botão A desliga os PWMs dos LEDs vermelho e azul.<br>
No display há um quadrado que acompanha o movimento do Joystick.
